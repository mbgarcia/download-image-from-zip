### Descrição do serviço


A aplicação possui um serviço REST que disponibiliza uma chamada para recuperar uma imagem de um arquivo compactado.

O arquivo compactado é recuperado via FTP, e armazenado temporariamente em uma pasta com permissões de leitura/escrita pela aplicação.

Use o parametro ZIP_FOLDER para informar esta pasta temporária.
